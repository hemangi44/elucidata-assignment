Assignment for Elucidata.
How to run the project?
1. Clone the url into local folder.
2. Create virtual environment for python using-
	virtualenv venv
3. Activate the 'venv' environment using-
	venv\Scripts\activate
4. Install the python dependencies using-
	pip install -r requirements.txt
5. Run the project using-
	streamlit run main.py
6. When website is loaded on browser there will be 2 options-
	1. Data Lookup
	2. Data Plot
7. Select one option at a time to play with data
